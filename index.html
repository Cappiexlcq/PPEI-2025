Mini-projet : Analyse d’URLs en Bash

Génération automatique d'un tableau HTML contenant : codes HTTP, encodages et nombre de mots.

Présentation

Ce mini-projet permet d’analyser automatiquement une liste d’URLs. Le script Bash récupère :

  • le code HTTP
  • l’encodage de la page
  • le nombre de mots du contenu

Les résultats sont ensuite compilés dans une page HTML sous forme de tableau.
